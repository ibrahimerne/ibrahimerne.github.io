Alternatif: Sublime içinde "Preview" eklentisi veya VS Code'da Live Server kullanabilirsin. 7) Siteni internetten erişilebilir yapma — kolay yollar Aşağıdan birini seçebilirsin (hepsi ücretsiz seçenekler sunar): a) GitHub Pages (kolay, ücretsiz) Bir GitHub hesabı aç. Yeni bir repo oluştur: kullaniciadi.github.io (kullanıcı adı yerine GitHub kullanıcı adını koy). Proje dosyalarını repo'ya yükle (push). Repo ismi doğruysa site otomatik olarak https://kullaniciadi.github.io/ adresinde yayına geçer. b) Netlify (sürükle-bırak veya Git ile) Netlify'ye kaydol, siteni doğrudan repo ile bağla ya da index.html içeren klasörü sürükleyip bırak. Alan adı Netlify tarafından verilir; istersen özel domain bağlayabilirsin. c) Vercel Özellikle front-end projelerine uygun, repo ile bağlantı kurulup otomatik deploy sağlar. Not: Özel domain (ör. seninadim.com) almak istersen domain registrar'dan satın alıp DNS ayarlarıyla GitHub Pages/Netlify/Vercel'e yönlendirirsin (CNAME/A kaydı). 8) Git ile hızlı yayın (temel komutlar) Eğer bilgisayarında Git varsa: cd my-website git init git add . git commit -m "İlk sürüm" # GitHub'da repo oluşturduktan sonra: git remote add origin https://github.com/KULLANICIADI/REPO.git git branch -M main git push -u origin main 9) Sonraki adımlar / İyileştirme fikirleri Mobil uyumluluk (media queries) ekle. Ama web siteni yayınlamak için Repository (depo) oluşturman gerekiyor. Adım adım sana doğru yolu göstereyim: 1. Repository (Depo) Oluştur GitHub ana sayfana git. Sağ üstteki “+” işaretine tıkla → New repository seç. Depo ismini şu formatta ver: kullaniciadın.github.io (örnek: ibrahimerne.github.io) Bu isim çok önemli! Böyle yaparsan GitHub Pages otomatik olarak siteni bu adreste açar: https://ibrahimerne.github.io/ Repo’yu Public yap. İstersen README oluşturabilirsin ama zorunlu değil. Create repository butonuna bas. 2. Site Dosyalarını Yükle Senin elinde index.html, styles.css ve script.js dosyaları olacak. Yeni açtığın repo’da “Add file → Upload files” seç. Bilgisayarındaki dosyaları sürükle-bırak yap. Commit et (altta yeşil Commit changes tuşu var). 3. GitHub Pages Ayarı Repo’nun ayarlarına (Settings) git. Sol menüde Pages sekmesini bul.
